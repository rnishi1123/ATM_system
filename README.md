# ATM Client-Server System (Java)

## 概要

Javaで実装したクライアントサーバ型ATMシステムです。
TCPソケット通信を利用してクライアントとサーバ間で口座操作を行います。

サーバはクライアント接続ごとにスレッドを生成することで、複数クライアントからの同時接続に対応する構成になっています。
大学の演習課題として作成しました。

sever/clientのプログラムはそれぞれのファイルに入っています。

## 機能

* 口座開設
* ログイン認証
* 預金
* 払い出し
* 残高照会
* 取引終了

## マルチスレッド処理

サーバはクライアント接続ごとにスレッドを生成し、それぞれの接続を独立して処理します。

```java
Socket socket = server.accept();
FileRcvThread th1 = new FileRcvThread(socket);
th1.start();
```

`FileRcvThread` クラスが `Thread` を継承しており、クライアントごとのリクエスト処理を並行して実行します。

クライアント（ATM_Client.java）がTCPソケット通信でサーバ（ATM_Server.java）に接続し、口座操作のリクエストを送信する。サーバは接続ごとにスレッドを生成して処理を行い、口座情報はファイルとして保存・更新される。
